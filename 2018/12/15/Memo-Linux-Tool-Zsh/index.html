oh-my-zsh

Github | 参考博客进行安装

关于PS1环境变量的折腾 因为含特殊字符GitBook构建通不过,只能放出去了

  1. 安装好 zsh wget git
  2. sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
  3. vi ~/.zshrc 进行配置
